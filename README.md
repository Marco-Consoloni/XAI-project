## Project Overview
We adress domain-specific problems:

- Early detection of plant diseases during initial phases of infection
- Detection on non-homogenous and complex backgrounds;
- Detection in single and multiple overlapping leaves

We trained different models for plant disease classification from images and compare them using post-hoc XAI techniques, including: **Integrated Gradients (IntGrad)**, **LIME (Local Interpretable Model-agnostic Explanations)**, **Grad-CAM++**.
